09 / Builder sovereignty

Gas is a choice, not a tax on every click.

Gas는 세금이 아니다.
빌더가 선택하는 보안 레벨이다.

기본은 off-chain. 정산과 결제만 on-chain. UX와 비용을 빌더가 직접 통제한다.

Execution Policy Builder decides
Off-chain by default 앱 이동, 메모 저장, 권한 체크는 빠르고 저렴한 signed request path로 처리한다.
On-chain when money moves OKRW 결제, 주문 정산, 송금 proof는 maroo transaction으로 검증한다.
Cost stays under control 빌더가 UX, 보안, 비용의 경계를 직접 정한다. 그래서 gas 비용을 줄인다.
Use
chain
with intent
Principle: off-chain for interaction, on-chain for settlement and public proof
